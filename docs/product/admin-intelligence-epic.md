# EPIC — Admin Intelligence Dashboard

## Objetivo

Criar um dashboard administrativo para donos da plataforma, escolas e operadores acompanharem adoção, resultados de aprendizagem, riscos, qualidade pedagógica, uso de features e saúde operacional.

## Personas

```text
Platform Admin
School Admin
Teacher
Parent/Guardian
Engineering/Admin Operator
```

## Módulos

### ADM-001 — Admin Home Overview

KPIs:

```text
active students
active guardians
active teachers
active schools/classes
activities completed
average accuracy
study time
active gaps
resolved gaps
open action plans
weekly active students
monthly active students
```
